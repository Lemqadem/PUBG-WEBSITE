Découvrez les cartes de PUBG

PUBG propose une sélection variée de cartes, chacune avec ses particularités. Qu'il s'agisse de vastes zones ouvertes, de villes abandonnées ou de jungles denses, chaque environnement demande une approche différente. Cette page vous présente les principales cartes du jeu, avec leurs caractéristiques, leurs points forts et les styles de jeu qu'elles encouragent.

Erangel

  • Type : Classique
  • Taille : 8x8 km
  • Ambiance : Île soviétique fictive, avec des zones rurales et urbaines.
  • Description : Carte équilibrée avec de grandes villes, des zones forestières et des bases militaires. Idéale pour tous les styles de jeu.

Miramar

  • Type : Désertique
  • Taille : 8x8 km
  • Ambiance : Désert sec d'Amérique latine
  • Description : Terrain ouvert avec peu de couverture, favorise les combats à longue distance. Des villes comme Los Leones et El Pozo offrent de l'équipement de qualité.

Sanhok

  • Type : Jungle / Tropicale
  • Taille : 4x4 km
  • Ambiance : Forêt tropicale d'Asie du Sud-Est.
  • Description : Petite carte rapide et intense avec beaucoup de végétation. Favorise les combats rapprochés.

Vikendi

  • Type : Neige
  • Taille : 6x6 km
  • Ambiance : Paysage hivernal, ambiance européenne de l'Est.
  • Description : Paysage enneigé avec des bâtiments imposants. Idéal pour le jeu stratégique.

Taego

  • Type : Rurale / Corée
  • Taille : 8x8 km
  • Ambiance : Campagne coréenne moderne.
  • Description : Environnement varié entre villages et zones boisées. Très axée sur le jeu d'équipe.

Deston

  • Type : Urbain futuriste
  • Taille : 8x8 km
  • Ambiance : Ville abandonnée avec marécages et gratte-ciels
  • Description : Carte mêlant zones urbaines denses, toits accessibles, et terrains inondés. Gameplay varié et dynamique.
